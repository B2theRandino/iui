About Extended Styles

Extended styles is an extension for the iUI framework. It contains new CSS classes for buttons, lists, forms, texts and photos.
It has been developped and fully tested with iUI 0.4x.

Remi Grumeau

HTML code


  • <a href="#page1">Page 1</a>

Here is how you could show a link HTML code with the following look'n'feel

  • <div class="row">
  •   <p>
  •     <ul class="code">
  •     <li><a href="#page1">Page 1</a></li>
  •     </ul>
  •   </p>
  • </div>

We use ul and li so the line number renders well on multiple line.

Please use ' &lt; ' for ' < ' and ' &gt; ' for ' > ' so browser doesn't see it as an HTML element
