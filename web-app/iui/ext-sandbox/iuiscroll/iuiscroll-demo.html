About iUIScroll

iUIScroll is an implementation of iScroll library for iUI.

Just add a classname "iuiscroll" to the main div/ul/form of the page, and an id value to '_scroller' prefixed by the main id name

  • <div id="default" title="Default" class="iuiscroll">
  •   <div id="default_scroller">
  •     This text is scrollable
  •   </div>
  • </div>

  • <div id="music" title="Music Demo" class="iuiscroll">
  •   <div id="default_scroller">
  •     Album 1
  •   </div>
  • </div>

iui.iScroll.myScroll (iScroll object)
Default is null, set by activeScroller.

It is set on afterTransition by the function activeScroller() and use by setHeight().

To access all iScroll functions like destroy() or refresh() (see iScroll website for more infos).

Ex: "home_scroller", "music_scroller", ...

iui.iScroll.activeScroller (Function)
Loaded once on load, and after each 'afterInsert' iUI event

It selects all iUI screens (aka DOM element with id & title attributes) with a classname to 'iuiscroll', their child DOM elements with id attribute equal to parents ID + '_scroller'.

For each match, it adds an 'afterTransition' event listener that:
- set iScroll object height using setHeight()
- destroy existing iScroll objects
- declare a iScroll object on this DOM element

iui.iScroll.setHeight (Function)
Loaded on each 'afterTransition' iUI event on iScroll-enable screens
- page (string)

Use page, or iui.getSelectedPage() if empty.
Detect toolbar height, and if present in the DOM for this screen, footer height too. It then set height & min-height of the scroller based on window height - ( toolbar height + footer height).
