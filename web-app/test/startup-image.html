Startup Image

This script in the header switches between 'iui-startup-hd.png' for iPad, or uses 'iui-startup.png' for others.

  • <script type="application/x-javascript">
  • if(navigator.userAgent.match(/iPad/i) != null)
  •   document.write('<link rel="apple-touch-startup-image" href="iui/iui-startup-ipad.png">');
  • else
  •   document.write('<link rel="apple-touch-startup-image" href="iui/iui-startup.png">');
  • </script>
